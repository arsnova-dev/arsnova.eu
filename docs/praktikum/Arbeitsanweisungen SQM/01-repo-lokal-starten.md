# Arbeitsanweisung 01 - SQM

## Auftrag

Sorge bis zur nächsten Sitzung selbstständig dafür, dass das Projekt **lokal startbar** ist.

## Was du konkret machen sollst

- Richte die lokale Entwicklungsumgebung nach [onboarding.md](../../onboarding.md) ein.
- Führe das notwendige Setup eigenständig aus, bis das Projekt lokal läuft.
- Prüfe, dass die Anwendung im Browser erreichbar ist.

## Ergebnis für die nächste Sitzung

- Das Repo ist auf deinem Rechner eingerichtet.
- Du kannst den lokalen Start reproduzierbar erklären.
- Du kannst zeigen, dass die Anwendung lokal läuft.
- Du kannst die wesentlichen Setup-Schritte selbstständig benennen.
